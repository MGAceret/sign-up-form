# Sign-up-Form
A semi-responsive sign-up form project built as per challenge of The Odin Project curriculum.

## Features
- Semi-responsive Form Fields through form validation:
    - First Name (required)
    - Last Name (required)
    - Email (required)
    - Phone Number (11-digit validation)
    - Password (Minimum: 12-Character validation)
    - Confirm Password (Minimum: 12-Character validation)

## Programming Languages Used
- HTML
- CSS
